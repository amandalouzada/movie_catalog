# Catálogo de filmes

  ## Abordagem
  Projeto dividido em dois diretórios FrontEnd e BackEnd, onde serão realizadas entregas contínuas.
    - Estrutura Basica Front
    - Estrutura Basica BackEnd
    - Cadastro de Filmes
    - Listar/Editar
    - Autenticação
    - Busca na API OMDB
    
    
  ## Tecnologias
  
  - Angular
  - Bootstrap
  - Sass
  - Node.js
  - Express
  - Mongodb
  - JWT
  
  
  # Desafios

    Realizar todos os testes Front e Back
